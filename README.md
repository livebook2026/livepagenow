# 📚 livepagenow

Мини-сайт для презентации книг онлайн-площадки livepagenow

---

## 🚀 Быстрый старт

1. Откройте сайт по ссылке https://livebook2026.github.io/livepagenow/


---

## ✨ Функциональность

- ⭐ Рейтинг книги (сохраняется в localStorage)
- 📊 Прогресс-бар рейтинга
- 🏷️ Интерактивные теги-жанры
- 🛒 Покупка на ЛитРес
- 📖 Чтение фрагмента на ЛитРес
- 🔗 Возможность поделиться: ВК, ОК, Email
- 📋 Копирование ссылки
- 📱 ВК-сообщество
- 📋 Копирование описания
- ↑ Кнопка "Наверх"
- 🎬 Микро-анимации
- 🖼️ Ленивая загрузка изображений

---

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Animations, Variables)
- JavaScript (ES6+, localStorage)
- Inter Font
