1. Откройте сайт по ссылке []


## ✨ Функциональность

- **Рейтинг** — оценка книги со звёздами (сохраняется в localStorage)
- **Теги** — интерактивные жанры с визуальной обратной связью
- **Покупка** — переход на ЛитРес
- **Фрагмент** — чтение отрывка на ЛитРес
- **Шеринг** — публикация в ВКонтакте и Одноклассниках
- **ВК-сообщество** — ссылка на группу издательства
- **Ленивая загрузка** — изображения загружаются по мере необходимости
