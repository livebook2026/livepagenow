# 📚 livepagenow

Мини-сайт для презентации книг онлайн-площадки livepagenow

---

## 🚀 Быстрый старт

1. Откройте сайт по ссылке https://livebook2026.github.io/livepagenow/

---

## ✨ Функциональность

- ⭐ Рейтинг книги (сохраняется в localStorage)
- 📊 Прогресс-бар рейтинга
- 📅 Дата публикации
- 💰 Цена книги
- 🏷️ Интерактивные тэги-жанры
- 🛒 Покупка на ЛитРес
- 📖 Чтение фрагмента на ЛитРес
- 🔗 Шеринг: ВК, ОК, Email
- 📋 Копирование ссылки
- 📱 ВК-сообщество
- ↑ Кнопка "Наверх"
- 🎬 Микро-анимации
- 🖼️ Ленивая загрузка изображений
